Knuth Morris Pratt algorithm is a very useful algorithm in string computation.
This algorithm is faster than the naive one in resolving exact pattern matching because it first creates an index table on the pattern to find and then uses this index to scan the text in a more efficient way.

This web service has been made in order to find exact matching between a short nucleotide sequence provided by the user and the genome of SARS-COV-2.
